# Project Structure

```
Folder App : untuk menyimpan aplikasi kita, semua kode program kita disimpan di sini

Folder Bootsrap : laravel akan melakukan proses bootstraping pada awal aplikasi dijalankan, jadi jangan diubah dan dihapus

Folder Config : untuk menyimpan semua konfigurasi

Folder Database : untuk terkoneksi ke database dan bisa melakukan database migrasi, tetap jangan dihapus

Folder Lang : untuk internalization

Folder Public : untuk entry point atau diakses oleh publik

Folder Resources : untuk diakses oleh publik namun tidak ingin dilihat oleh publik

Folder Routes : untuk melakukan routing

Folder Storage : untuk menyimpan file, semisal log

Folder Test : untuk menyimpan unit test

Folder Vendor : bawaan dari composer
```
